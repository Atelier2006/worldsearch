# worldsearch

アスユイの VRChat お気に入りワールドを、カテゴリ選択と検索で辿るためのポータルサイト。

🔗 **公開URL**: https://atelier2006.github.io/worldsearch/

## 機能
- カテゴリ切り替え（第1弾: **展覧会系** 実装済み。パーティクル / 雑多 などは順次追加）
- ワールド名・タグ・説明・定員でのリアルタイム検索
- タグ絞り込みチップ、並び替え（人気 / 訪問数 / 名前 / 定員）
- 各ワールドの詳細（最大/推奨人数・お気に入り数・訪問数・説明欄・タグ・VRChatリンク）

## 構成
| ファイル | 役割 |
|----------|------|
| `index.html` | サイト本体（UI・検索・詳細モーダル） |
| `data.js` | ワールド／カテゴリのデータ。**ここを編集するだけで追加・更新できます** |

## ワールドの追加方法
`data.js` の `WORLDS` 配列にオブジェクトを追記し、`note`（説明欄）を書くだけです。
新カテゴリは `CATEGORIES` に1行足し、対象ワールドの `category` をその id に設定します。

---
データ出典: VRChat のお気に入りワールドより。サムネイルは VRChat にログイン済みのブラウザで表示されます。
